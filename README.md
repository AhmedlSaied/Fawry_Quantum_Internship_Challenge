# Fawry Quantum Car Challenge

A simple car factory simulation built with C# as a solution for Fawry's internship challenge.

## What it does
- Creates cars with different engine types (Gas, Electric, Hybrid)
- Controls car speed (accelerate, brake, start, stop)
- Allows replacing the engine of an existing car

## Engine Types
- **GasolineEngine** – runs on gas
- **ElectronicEngine** – runs on electricity
- **MixedHybridEngine** – uses electric below 50 km/h, gas above

## Tech
- C# / .NET
- OOP principles (Interface, Inheritance, Polymorphism)
